# VARYNX 2.0

VARYNX 2.0 is a privacy-first, offline security platform that protects all your devices through a local encrypted mesh.
No cloud. No accounts. No tracking. Your data never leaves your devices.

## What VARYNX Does
- Detects phishing, malicious apps, overlays, DNS tampering, USB threats, and suspicious behavior
- Analyzes patterns using built-in intelligence modules
- Responds automatically with reflex actions
- Syncs threat signals across your devices through encrypted peer-to-peer mesh
- Works fully offline across Android, Windows, Linux, WearOS, and embedded nodes

## Key Features
- Multi-device offline mesh
- Local threat detection
- Reflex engine
- Intelligence modules
- Desktop dashboard
- Android guardian app
- Zero cloud dependency
- Zero telemetry

## Platforms
- Android
- Windows Desktop
- Linux
- WearOS
- HomeHub
- Pocket Node
- Satellite Node
